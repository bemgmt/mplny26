# Backgrounds

This folder contains background images that can be used in photo compositions.

## Background Structure

Each background should have:
- A full-size image (e.g., `bg-1.png`)
- A thumbnail image (e.g., `bg-1-thumb.png`)

## Adding New Backgrounds

1. Add the background images to this folder
2. Update `lib/js/config.ts` with the new background information
3. Ensure images are optimized for web use
