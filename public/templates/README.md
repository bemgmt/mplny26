# Templates

This folder contains photo template images that can be applied to captured photos.

## Template Structure

Each template should have:
- A full-size image (e.g., `template-1.png`)
- A thumbnail image (e.g., `template-1-thumb.png`)

## Current Templates

1. **template-1** - Classic Frame
2. **template-2** - Dragon Border
3. **template-3** - Lantern Frame
4. **template-4** - Calligraphy Style
5. **template-5** - Modern Minimalist

## Adding New Templates

1. Add the template images to this folder
2. Update `lib/js/config.ts` with the new template information
3. Ensure images are optimized for web use
